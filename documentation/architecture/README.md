# Architektur

Dieser Ordner fasst die Architektur von **pflegeleicht.online** zusammen: wer mit dem System interagiert, aus welchen Bausteinen es besteht, wie Anfragen zur Laufzeit ablaufen und welche Entscheidungen dabei getroffen wurden. Die Sichten ergänzen sich von außen nach innen und sind für Entwicklung, Onboarding und Reviews gedacht.
